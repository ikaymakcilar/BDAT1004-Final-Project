Get Items in Range:

GET /api/items/range?start=YYYY-MM-DD&end=YYYY-MM-DD

Retrieves items within the specified date range.

            http://127.0.0.1:8000/api/items/range?start=2023-11-25&end=2023-11-26

Get Item by ID:

GET /api/items/<id>

Retrieves an item by its ID.

Example Request:

            http://127.0.0.1:8000/api/items/656a8a89808d7238d6de0651

Example Output:

            {
                "_id": "656a8a89808d7238d6de0651",
                "date": "Sat, 02 Sep 2023 22:00:14 GMT",
                "price": 25863.6019089
            }
